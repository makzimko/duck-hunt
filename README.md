# duck-hunt

## Getting started

Clone repository and install dependencies
```shell
git clone https://github.com/makzimko/duck-hunt.git
cd duck-hunt
npm ci
```
Run application and visit http://localhost:5173/:
```shell
npm run dev
```

## Functionality

### Already done
- server generates randomly information for new target: direction, delay before starting and offset
- front count score: hits and total targets
- animated duck with audio for shot and hit
- enjoy and crying state for hunting dog
